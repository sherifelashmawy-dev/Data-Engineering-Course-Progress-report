## About This Repository

This repository contains my solutions for the course exercises (0.1 through 18). Each exercise is in its own folder with code, documentation, and any required files.

## Setup

### Requirements
- Python 3.11
- Visual Studio Code
- Docker Desktop
- Google Cloud Account

### Installation
```bash
# Create conda environment
conda create -n data-eng python=3.11
conda activate data-eng

# Install common dependencies
pip install pandas numpy matplotlib jupyter
```
